A simple blockchain implementation built in Rust for learning purposes.

---

## 🚀 Features

* 🔗 Block linking using hashes
* 🧱 Genesis block creation
* 📦 Simple blockchain structure
* ⚡ Fast and lightweight (Rust)

---

## 🛠️ Built With

* 🦀 Rust
* 📦 Cargo

## ▶️ How to Run

```bash
git clone https://github.com/your-username/stellar-blockchain.git
cd stellar-blockchain
cargo run
```

---

## 🧠 How It Works

Each block contains:

* Index
* Timestamp
* Data
* Previous Hash
* Hash

Blocks are linked together forming a secure chain 🔐

---
